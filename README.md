## 🚀 Como Deploy no Render

### 1. Criar uma conta no GitHub
- Acesse [github.com](https://github.com)
- Cadastre-se se não tiver uma conta

### 2. Criar um repositório no GitHub
- Clique em **"New repository"**
- Nomeie como: `joao-linaldo-landing`
- Escolha **Public** ou **Private** (de acordo com sua preferência)
- Não inicialize com README (já temos um)
- Clique em **"Create repository"**

### 3. Enviar o código para o GitHub
Abra o Git Bash ou Terminal na pasta do projeto e execute:

```bash
# Inicializa o git
git init

# Adiciona todos os arquivos
git add .

# Primeiro commit
git commit -m "Initial commit - Landing Page João Linaldo"

# Conecta ao repositório remoto (substitua SEU-USUARIO)
git remote add origin https://github.com/SEU-USUARIO/joao-linaldo-landing.git

# Envia para o GitHub
git branch -M main
git push -u origin main
```

### 4. Deploy no Render
1. Acesse [render.com](https://render.com)
2. Crie uma conta ou faça login
3. Clique em **"New +"** → **"Static Site"**
4. Conecte sua conta do GitHub
5. Selecione o repositório `joao-linaldo-landing`
6. Configure:
   - **Name**: `joao-linaldo`
   - **Branch**: `main`
   - **Publish directory**: Deixe vazio (root)
7. Clique em **"Create Static Site"**

### 5. Pronto! 🎉
Seu site estará online em minutos em:
`https://joao-linaldo.onrender.com`
